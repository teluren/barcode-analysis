## Кратко

- Все три билета — **CODE-128** (стандартный линейный штрихкод)
- Билеты №1 и №2 содержат **по два CODE-128**, билет №3 — **один**
- Маршрут: **пос. Александровская** ↔ **Балтийский вокзал** (СПб)
- Билет №1 — в прямом направлении, №2 и №3 — обратно
